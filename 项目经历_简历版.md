# 项目经历（简历版）

## 校园预约管理系统 | Java全栈开发工程师

**项目时间**：2024.03 - 至今  
**技术栈**：Spring Boot 3.5 + MyBatis-Plus + MySQL + Redis + JWT + Vue 3 + TypeScript + Element Plus  
**项目规模**：个人独立开发，20+功能模块

### 项目描述
基于Spring Boot 3的校园服务预约平台，支持用户注册登录、服务预约、角色权限管理等核心功能。采用前后端分离架构，集成JWT认证、邮件服务、文件存储、AI大模型等企业级特性，为校园用户提供便捷的服务预约体验。

### 核心职责与技术实现

#### 1. 系统架构设计与代码实现
- 基于分层架构思想设计项目结构，将Controller、Service、Mapper、Entity分层组织
- 实现`api/thirdparty`包结构，统一管理第三方服务（AI大模型、文件存储OSS、天气API）的集成
- 设计`vo/response`响应对象体系，规范接口返回格式，提升前后端协作效率
- 引入`RestTemplateConfig`配置类，实现HTTP客户端的统一管理与连接池配置

#### 2. 安全认证体系构建
- 实现基于JWT的用户认证机制，完成注册、登录、密码重置全流程
- 设计RBAC权限模型，支持普通用户与管理员角色切换
- 实现邮箱验证码登录、图形验证码防刷、密码加密存储（BCrypt）
- 配置Spring Security，实现接口级别的权限控制与Token验证

#### 3. 性能优化与高并发处理
- 使用Redis缓存验证码、用户信息等热点数据，减少数据库查询压力
- 实现Redis分布式锁，解决并发预约场景下的数据一致性问题
- 配置HikariCP连接池，优化数据库连接性能
- 设计缓存预热策略，提升系统启动后的响应速度

#### 4. 企业级功能实现
- 集成大模型API（DeepSeek、Qwen），实现智能客服与对话功能
- 实现文件上传功能（OSS），支持图片、文档等多种文件类型
- 集成第三方天气API，为用户提供实时天气信息
- 配置全局异常处理器（@RestControllerAdvice），统一错误响应格式

#### 5. 工程化能力建设
- 集成MyBatis-Plus代码生成器，提升数据库操作开发效率
- 使用SpringDoc生成OpenAPI文档，配合Knife4j提供可视化API界面
- 实现邮件服务集成，支持验证码发送、预约通知等场景
- 配置日志系统，使用SLF4J记录关键业务操作
